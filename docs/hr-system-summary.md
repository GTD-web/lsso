## 🔧 **핵심 기능**

### 📊 **워크로드 관리**

```sql
-- 과부하 직원 자동 감지
SELECT name, SUM(투입률) FROM 프로젝트배정
WHERE 상태 = '활성' GROUP BY 직원ID
HAVING SUM(투입률) > 100%;
```

### 🎯 **스마트 배정**

-   ✅ 가용 용량 자동 체크
-   🚨 과부하 알림 발생
-   📈 실시간 대시보드

### 📈 **완전한 이력 추적**

-   🏢 부서 이동 기록
-   📊 승진 과정 추적
-   🚀 프로젝트 참여 내역
